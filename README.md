# Headphones Landing Page

A fully responsive landing page for headphones, designed using HTML and CSS.

---

## Features

- **Responsive Design**: Adapts to screens 480px and smaller.
- **Hover/Active States**: Links (`#FF6565`), buttons (opacity `0.9`).
- **Custom Fonts/Icons**: Source Sans Pro, Spin Cycle OT, and Holberton School icons.
- **Sections**: Header, "What we do...", "Our results", Contact Us, Footer.
- **Animations**: Added to enhance user experience.
