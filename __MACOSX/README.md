# Headphones Website

## Description
A modern and responsive web page for showcasing headphones with a professional layout and seamless mobile support.

## Features
- Responsive design switches to mobile-friendly layout at 480px.
- Links hover/active color: `#FF6565`.
- Buttons hover/active opacity: `0.9`.
- Content centered with max width of 1000px.

## Project Structure
```
alx_html_css/
├── headphones/
   ├── index.html

   └── images/
       ├── logo.png
       └── background.jpg
```

## How to Use
1. Clone the repo:
   ```bash
   git clone https://github.com/yourusername/alx_html_css.git
   ```
2. Open `index.html` in your browser.

## Interactions
- Links: Hover/active color `#FF6565`.
- Buttons: Hover/active opacity `0.9`.
